# H D IMPEX Catalogs Directory

This directory contains the PDF catalogs for H D IMPEX products.

## Catalog Files:

1. **agricultural-products.pdf** - Agricultural Products Catalog
   - Spices, grains, rice, and agricultural commodities
   - Complete specifications and pricing

2. **industrial-materials.pdf** - Industrial Materials Catalog  
   - Raw materials, chemicals, and industrial supplies
   - Technical specifications and applications

3. **consumer-goods.pdf** - Consumer Goods Catalog
   - FMCG products, electronics, and consumer items
   - Product features and availability

## Usage:

- Replace placeholder URLs in catalog.js with actual PDF file paths
- Ensure PDF files are optimized for web viewing
- Update catalog content regularly

## File Structure:
```
catalogs/
├── agricultural-products.pdf
├── industrial-materials.pdf
├── consumer-goods.pdf
└── README.md (this file)
```

## Notes:

- PDF files should be optimized for web viewing (compressed)
- Recommended file size: Under 10MB per catalog
- Use high-quality images but optimize for loading speed
- Include bookmarks for easy navigation within PDFs
